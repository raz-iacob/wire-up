# Changelog

Release notes for Wire-Up. Each release is a `## vX.Y.Z` section; the updater shows the sections newer than the installed version on **Settings → Updates**.

## v0.1.0 — 2026-07-08

- Initial release: pages, records with custom content types, media library, menus, categories, users and roles, inbox, integrations (Pexels, Google Analytics, Google Maps, Slack), analytics dashboard, SEO layer, multi-language support.
- Server install (`wireup:install`) and self-update mechanism (`wireup:update`, Settings → Updates, auto-updates).
